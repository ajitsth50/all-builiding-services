🏗️ All Building & Property Services Website
A modern, fast, and responsive website for Sydney-based remedial building and property maintenance specialists. Built with React and Vite for optimal performance, it showcases services, projects, and client trust while driving inquiries through intuitive CTAs.
Hero Preview 
🚀 Tech Stack

React — Dynamic, component-based UI
Vite — Fast bundling and HMR
React Router — Client-side routing
Tailwind CSS — Responsive, utility-first styling
ESLint — Code linting and quality
Deployed on — Vercel/Netlify (CI/CD ready)

📦 Features

Home — Hero banner, services overview, achievements stats, and dual CTAs
About — Company story, team expertise, and trust indicators
Services — Detailed breakdowns (e.g., Remedial Repairs, Plumbing, Fire Safety)
Projects — Image gallery/portfolio
Contact — Form, map, and quick contacts
Terms & Privacy — Dedicated legal pages
Get A Quote — Modal form for instant leads
Fully responsive, SEO-optimized, with animated stats and smooth navigation

🔧 Getting Started
1. Clone the Repo
Bashgit clone https://github.com/yourusername/all-building-property-services-website.git
cd all-building-property-services-website
2. Install Dependencies
Bashnpm install
3. Run Development Server
Bashnpm run dev
Open http://localhost:5173 to view.
4. Build for Production
Bashnpm run build
Output in /dist — deploy to Vercel, Netlify, or your server.
5. Customize

Update content in src/components/ (e.g., Hero.jsx, Services.jsx)
Replace placeholders with real images/stats
Add env vars for forms (e.g., Web3Forms API key)
